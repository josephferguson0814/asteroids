# Asteroids

This was part of a guided project on boot.dev to build the game Asteroids! Some of the code was written for me, but I had to work through a lot myself too!

## Features

- Takes user input to control their sprite in the game
- Randomly spawns in asteroid sprites that cause the game to end if the player collides with them
- Lets the user shoot at the asteroids, but if they're hit then the asteroids split

## Requirements

- Python 3.10 +
- Pygame package

## How to Run

1. Clone the repo: git clone https://github.com/josephferguson0814/asteroids
2. Navigate to the project: cd <path_to_project_repo>
3. Run the program: python3 main.py

## What I learned

- How to use Pygame to render things and take player input
